## Create a virtual environment

```bash
python -m venv .envr
```

## Activate the virtual environment
```bash
.envr\Scripts\activate.bat
#or
.\op a
```

## Verify that the virtual environment
```bash
python --version
```

## To deactivate the virtual environment
```bash
.venv\Scripts\deactivate
#or
.\op d
```
